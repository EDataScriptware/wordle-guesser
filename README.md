# Wordle Guesser
## To Run Wordle Guesser
*Assuming you have pipenv installed in your workspace...*
1. `pipenv install` 
2. `pipenv shell`
3. `run main.py`

## Guessing Words
```
1 Letter: `e`
2 Letter: `g`
3 Letter: `h`
```

returns 

```
['weigh', 'hedge', 'hinge', 'hodge', 'hague', 'helga', 'heigh', 'hagen', 'eight', 'hager', 'ghent', 'leigh', 'phage', 'neigh']
```
